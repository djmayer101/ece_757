Detailed Description

An address mapper changes the packet addresses in going from the slave port side of the mapper to the master port side.

When the slave port is queried for the address ranges, it also performs the necessary range updates. Note that snoop requests that travel from the master port (i.e. the memory side) to the slave port are currently not modified.
