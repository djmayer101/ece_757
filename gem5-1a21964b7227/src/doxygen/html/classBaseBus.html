Detailed Description

The base bus contains the common elements of the non-coherent and coherent bus.

It is an abstract class that does not have any of the functionality relating to the actual reception and transmission of packets, as this is left for the subclasses.

The BaseBus is responsible for the basic flow control (busy or not), the administration of retries, and the address decoding.
