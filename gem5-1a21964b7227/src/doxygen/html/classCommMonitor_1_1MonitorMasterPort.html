Detailed Description

This is the master port of the communication monitor.

All recv functions call a function in CommMonitor, where the send function of the slave port is called. Besides this, these functions can also perform actions for capturing statistics.

bool CommMonitor::MonitorMasterPort::isSnooping (  )  const [inline, protected, virtual]

Determine if this master port is snooping or not.

The default implementation returns false and thus tells the neighbour we are not snooping. Any master port that wants to receive snoop requests (e.g. a cache connected to a bus) has to override this function.

Returns:
true if the port should be considered a snooper
