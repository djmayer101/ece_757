Detailed Description

Simple class to hold onto a list of pairs, each pair having a memory instruction's sequence number and effective addr.

This list can be used for memory disambiguation. However, if I ever want to forward results, I may have to use a list that holds DynInstPtrs. Hence this may change in the future.
