Detailed Description

IsaFake is a device that returns, BadAddr, 1 or 0 on all reads and rites.

It is meant to be placed at an address range so that an mcheck doesn't occur when an os probes a piece of hw that doesn't exist (e.g. UARTs1-3), or catch requests in the memory system that have no responders..
