uint32_t MemoryVector::collatePages ( uint8_t *&  raw_data  )  [inline]

Function for collating all the pages of the physical memory together.

In case a pointer for a page is NULL, this page needs only a single byte to represent that the pointer is NULL. Otherwise, it needs 1 + PAGE_SIZE bytes. The first represents that the page pointer is not NULL, and rest of the bytes represent the data on the page.

void MemoryVector::populatePages ( uint8_t *  raw_data  )  [inline]

Function for populating the pages of the memory using the available raw data.

Each page has a byte associate with it, which represents whether the page was NULL or not, when all the pages were collated. The function assumes that the number of pages in the memory are same as those that were recorded in the checkpoint.
