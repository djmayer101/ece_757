Detailed Description

template<class Impl>
class OzoneCPU< Impl >

Light weight out of order CPU model that approximates an out of order CPU.

It is separated into a front end and a back end, with the template parameter Impl describing the classes used for each. The goal is to be able to specify through the Impl the class to use for the front end and back end, with different classes used to model different levels of detail.
