Detailed Description

The PioPort class is a programmed i/o port that all devices that are sensitive to an address range use.

The port takes all the memory access types and roles them into one read() and write() call that the device must respond to. The device must also provide getAddrRanges() function with which it returns the address ranges it is interested in.
