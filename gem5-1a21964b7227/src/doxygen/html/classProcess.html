Map a contiguous range of virtual addresses in this process's address space to a contiguous range of physical addresses.

This function exists primarily to enable exposing the map operation to python, so that configuration scripts can set up mappings in SE mode.

Parameters:
vaddr The starting virtual address of the range.
paddr The starting physical address of the range.
size The length of the range in bytes.
Returns:
True if the map operation was successful. (At this point in time, the map operation always succeeds.)
