Detailed Description

Implements a simple scoreboard to track which registers are ready.

This class assumes that the fp registers start, index wise, right after the integer registers. The misc. registers start, index wise, right after the fp registers.

Todo:
: Fix up handling of the zero register in case the decoder does not automatically make insts that write the zero register into nops.
