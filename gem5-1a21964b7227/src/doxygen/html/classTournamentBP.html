Detailed Description

Implements a tournament branch predictor, hopefully identical to the one used in the 21264.

It has a local predictor, which uses a local history table to index into a table of counters, and a global predictor, which uses a global history to index into a table of counters. A choice predictor chooses between the two. Only the global history register is speculatively updated, the rest are updated upon branches committing or misspeculating.
