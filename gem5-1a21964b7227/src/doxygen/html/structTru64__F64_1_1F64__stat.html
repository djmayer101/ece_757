Detailed Description

Stat buffer.

Note that Tru64 v5.0+ use a new "F64" stat structure, and a new set of syscall numbers for stat calls. On some hosts (notably Linux) define st_atime, st_mtime, and st_ctime as macros, so we append an X to get around this.
